# Follow The Flow

A memory and reaction-based game where players must replicate increasingly longer sequences of directional inputs. Challenge your memory skills and see how far you can go!

## Game Overview

Follow The Flow is a modern take on the classic Simon Says game. Watch the sequence of arrow directions light up with glowing effects, then repeat it perfectly. With each successful round, the sequence grows longer, making the game progressively more challenging.

## How to Play

1. **Start the Game**: Click "Start Game" or press Enter to begin
2. **Watch**: Observe the sequence of directions (Up, Right, Down, Left) as they light up with visual effects
3. **Repeat**: Click the corresponding buttons or use arrow keys to replicate the exact sequence
4. **Progress**: Successfully complete a round to advance to the next one with a longer sequence
5. **Game Over**: Make a mistake and the round ends - restart and try again!

## Controls

- **Arrow Keys**: Input directional commands (Up, Right, Down, Left)
- **Mouse Click**: Click the colored buttons to input directions
- **Enter**: Start a new round or begin a new game
- **R**: Restart the current game
- **Return to Lobby**: Go back to the start screen

## Game Features

- **Progressive Difficulty**: Each round adds one more direction to memorize
- **Visual Feedback**: Glowing colored buttons provide clear visual indication of the sequence
- **Round Tracking**: Display shows your current round number
- **Keyboard Support**: Full keyboard control for seamless gameplay
- **Responsive Design**: Play on desktop with full-screen experience
